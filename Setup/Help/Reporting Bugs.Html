Reporting Bugs

448Delta has been tested on Windows 98, Me, NT 4.0, & W2k.

Windows 95 kernel does not have sufficient threading support to run 448delta, therefore 448delta will not run on a windows 95 platform.

In the past releases of 448Delta few bugs have been reported but all found have been fixed. If 448Delta crashes and gives you an error, The error message should be followed by an error code (Error: xx). Please send the error code, a detailed description of what you were doing when the error occurred, what operating system you are using to: Tech@NetWorkDLS.Com.

Also, if you notice an error or a system malfunction that seems to be originating from 448Delta please send a detailed description of the problem and why you think its originating from 448Delta to: Info@NetWorkDLS.Com.

Please include you name in the e-mail so we can address you respectfully.

Allow atleast 24 - 72 hours for a response.
