448Delta PriorityBoost

448Delta allows you to enable or disable its Windows NT & 2000 PriorityBoost.

- 448Delta Priority Utility -

The above utility was installed with 448Delta and will be in the start menu under the group 448Delta was installed to. Default: Start > Program > NetWorkDLS > 448Delta

ProcessPriorityBoost

ProcessBoost enables Windows NT & 2K to temporarily boost the priority of the entire process.

When a thread is running in one of the dynamic priority classes, Windows NT temporarily boosts the process's priority when it is taken out of a wait state. This setting affects all existing threads and any threads subsequently created by the process.

ThreadPriorityBoost

ThreadBoost enables Windows NT & 2K to temporarily boost the priority of a single thread.

When a thread is running in one of the dynamic priority classes, Windows NT temporarily boosts the thread’s priority when it is taken out of a wait state.

Note: ProcessPriorityBoost & ThreadPriorityBoost settings are not set untill 448Delta is actually writing to the file.
