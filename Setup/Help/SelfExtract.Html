Creating Self-Extracting Executables

Self-Extracting Executables are actual programs that contain encrypted data that can be extracted securely.


To create Self-Extracting Executables simply click the "Create Self Extractor" check box on the password dialog of 448Delta.

To create Self-Extracting Executables from files or folders that have already been encrypted by 448Delta simply right click on any file encrypted by 448Delta and Select "Create Self Extractor" from the context menu. 448Delta will then pack all the data into an executable program that will decrypt all its contained data upon entering the correct password.

To decrypt Self-Extracting Executables, Simply open it (By hitting enter or double clicking on its icon). It will prompt you for the correct password. Enter a password and click ok.

Above is the 448Delta Self-Append dialog. It appears and displays the status of creating self-extracting executables when you check the "Create Self Extractor" box on the password dialog of 448Delta or after you right click on any file encrypted by 448Delta and Select "Create Self Extractor" from the context menu. If you wish to delete the source file after creating a self-extracting executable click the "Delete Source" button on the Self-Append dialog after it has finished.
